# Week 1 - Vehicle Management System

Inheritance + polymorphism project.

### What it does
- Create Car / Bike / Truck with specific attributes
- Start, stop, accelerate, brake, refuel
- Honk, headlights, trunk, wheelie, sidecar, cargo loading…
- Full interactive console menu

### Classes
- `Vehicule` (abstract)
- `Car`, `Bike`, `Truck` → concrete subclasses
- Polymorphism everywhere

### How to run
Just run `VehicleManagementApp.java` → menu with 13 options
